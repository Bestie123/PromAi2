# Development Plans

Эта папка содержит планы разработки проекта PromAi PKM.

## Файлы

- **development_plan.json** - Полный план разработки в JSON формате
- **DEVELOPMENT_PLAN_GUIDE.md** - Подробная документация по плану

## Структура плана

План разбит на 5 уровней:

- **Уровень 0 (MVP):** 68-94 дня - 29 критических критериев
- **Уровень 1:** 90-120 дней - Core UX и "магия"
- **Уровень 2:** 120-180 дней - Продуктивность и рост базы
- **Уровень 3:** 90-150 дней - Безопасность и плагины
- **Уровень 4:** 180-270 дней - Nice-to-have функции

**Общая длительность:** 548-814 дней (18-27 месяцев)

## Использование

Читайте `DEVELOPMENT_PLAN_GUIDE.md` для понимания структуры и приоритетов.
