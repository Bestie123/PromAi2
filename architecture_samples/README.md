# Architecture Samples - Refactored

Рефакторенные варианты архитектурной визуализации с автоматической генерацией координат.

## Файлы

### nodes_data.json
- **Без координат** - только данные узлов
- Координаты генерируются автоматически при загрузке
- Легко редактировать и добавлять новые узлы

### layout_generator.js
- Модуль автоматической генерации координат
- Алгоритм размещения:
  - Вопросы: горизонтально по 5 в ряд
  - Ответы: под вопросами с равномерным распределением
- Параметры:
  - `HORIZONTAL_SPACING`: 1000px между вопросами
  - `VERTICAL_SPACING`: 500px между рядами
  - `QUESTION_Y`: 100px начальная позиция вопросов
  - `ANSWER_Y`: 300px смещение ответов от вопросов

### architecture_auto.html
- Полная визуализация с автоматическим layout
- Загружает `nodes_data.json`
- Генерирует координаты на лету
- Все функции оригинала: zoom, pan, connections

## Преимущества

1. **Простота редактирования** - не нужно вручную задавать координаты
2. **Масштабируемость** - легко добавлять новые узлы
3. **Консистентность** - единый алгоритм размещения
4. **Читаемость** - JSON содержит только смысловые данные

## Использование

```bash
# Запуск локального сервера
cd architecture_samples
python -m http.server 8000
# Открыть http://localhost:8000/architecture_auto.html
```

## Добавление узлов

```json
{
  "id": "new-node",
  "type": "question",
  "title": "❓ Новый вопрос?",
  "description": "Описание",
  "details": "Детали",
  "children": ["answer1", "answer2"],
  "criteriaId": 30
}
```

Координаты будут рассчитаны автоматически.
